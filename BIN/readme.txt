2010.7.8.1
---------
  모든 INI설정값을 파라미터로 받을 수 있도록 함. 
    예를 들어 다음과 같이 해주면
      dkw "chdir=C:\"
    INI에 지정된 값을 무시하고 C:\ 를 기준으로 실행된다. 

  fontSizeCtrl 이 켜져있을 때 컨트롤+마우스 휠로 폰트크기 조절되게 함. 

  http://sites.google.com/site/craftware/ckw/download 의 코드 적용. 
    윈도7, 비스타에서 초기실행시 작은 콘솔창 나오는 문제 수정. 
    타이틀바를 우클릭할 때 나타나는 시스템 메뉴에 "Copy All" 기능 추가. 
    설정 추가
      SelectOverScreen=yes  : 마우스 드래그로 선택할 때 화면을 벗어난 영역까지 선택하는 기능. 


2010.7.7.2
----------
  설정에 다음 항목이 추가됨. 

    fontSizeCtrl=yes    : 컨트롤키와 숫자패드의 +/- 키를 써서 폰트크기를 조절한다.
    fontSizeCtrlStep=2  : 위의 기능을 사용할 때 크기조절 단계를 설정한다. 

    UseCtrl_C_Copy=yes  : 컨트롤+C 키를 사용해 클립보드에 복사한다. 마우스 복사는 무시됨. 
    UseCtrl_V_Paste=yes : 컨트롤+V 키를 사용해 클립보드 내용을 붙여넣기 한다.
